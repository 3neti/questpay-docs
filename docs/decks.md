# The Three Decks

---

## Overview

QuestPay™ requires different messaging for different audiences. We've developed three specialized pitch decks, each tailored to specific stakeholder priorities.

---

## Sponsor Edition

### Target Audience

- Banks and digital wallets
- Major merchants and retailers
- FMCG and consumer brands
- Telecommunications companies

### Key Emphasis

**Brand ROI & Engagement**

- Transaction attribution and measurement
- User acquisition metrics
- Brand exposure quantification
- Social media amplification
- Competitive positioning

### Core Messages

1. **Measurable Impact**: Direct attribution of transactions to show participation
2. **Youth Engagement**: Access to digitally-savvy demographic
3. **Innovation Association**: Position brand as forward-thinking
4. **Content Marketing**: Authentic brand integration vs. traditional ads
5. **Data Insights**: User behavior and preference data

### Deck Structure

1. Market opportunity
2. Format overview (2-3 slides)
3. Sponsorship packages and pricing
4. Brand integration examples
5. ROI metrics and measurement
6. Success stories / case studies
7. Next steps and timeline

**Length**: 15-20 slides  
**Tone**: Business-focused, ROI-driven

---

## Ownership Edition

### Target Audience

- Celebrities and influencers
- Media producers and production companies
- Investors and venture capitalists
- Entertainment industry executives

### Key Emphasis

**Format Rights, Equity & Franchise Model**

- IP ownership and protection
- Revenue share structures
- Franchise licensing potential
- Equity participation options
- Celebrity/influencer involvement opportunities

### Core Messages

1. **Proven Format**: Reality competition + payments = tested formula
2. **Scalable IP**: Franchiseable across markets and themes
3. **Multiple Revenue Streams**: Not just ad-dependent
4. **Celebrity Opportunities**: Host, judge, or producer roles
5. **Growth Potential**: Pilot → Season → Franchise → Global

### Deck Structure

1. Format overview and hook
2. Market size and opportunity
3. Competitive landscape
4. IP and ownership structure
5. Revenue model and projections
6. Franchise roadmap
7. Team and track record
8. Investment ask and use of funds

**Length**: 20-25 slides  
**Tone**: Investment-pitch style, growth-focused

---

## Tech-Backbone Edition

### Target Audience

- Fintech partners and wallet providers
- Internal R&D and engineering teams
- Technology vendors and platforms
- Regulatory and compliance teams

### Key Emphasis

**Architecture, API Flow & Data Integrity**

- System architecture and scalability
- API integration requirements
- Security and compliance
- Data flow and verification
- Technical feasibility

### Core Messages

1. **Built on x-Change**: Proven voucher platform
2. **API-First Design**: Easy integration with existing wallets
3. **Real-Time Verification**: Instant transaction confirmation
4. **Compliance-Ready**: KYC/AML built-in
5. **Scalable Architecture**: Handle 100k+ concurrent users

### Deck Structure

1. System architecture overview
2. Core components and technology stack
3. API integration guide
4. Security and compliance framework
5. Data flow diagrams
6. Scalability and performance
7. Development roadmap
8. Integration timeline and support

**Length**: 25-30 slides  
**Tone**: Technical, detailed, architecture-focused
